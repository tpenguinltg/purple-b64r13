# purple-b64r13
A simple libpurple plugin for encoding sent messages through base64-rot13

## Building and Installation

1. Execute

        make
        make install

   (_Binary releases to come_)

2. Activate the plugin from Pidgin's plugin dialog (Tools > Plugins)

## Usage

To encode:
> /b64r13 Message

To decode:
> /b64r13d GJImp2SaMDb=
